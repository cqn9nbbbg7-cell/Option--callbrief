# Option CallBrief

Automated call option screening with Excel export and optional Discord alerts.

## What it does
- Pulls call option chains via yfinance
- Computes liquidity, spreads, breakeven, and move-needed %
- Scores contracts for research priority
- Exports clean Excel reports

## Usage
```bash
python callbrief.py
```

## Disclaimer

Research tool only. Not financial advice.
